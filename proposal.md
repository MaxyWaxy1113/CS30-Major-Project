# Major Project Proposal

## description
Monkeytype-like typing tester

## Needs to Have List
- Start Screen
- randomly generated set of words
- user input feedback(red letters when letter is typed wrong, grey letters when a letter is typed right)
- End display of words per minute and overrall input accruacy
- Choice of timed test (15 sec, 30 sec, 45 sec, 1 minute)
- The ability to restart the test if tab and enter are clicked together

## Nice to Have List
- Background music
- Ability to change colour scheme
- Different languages (English, French, Spanish)
- A mode that includes puntuation and numbers
- Scoring system
